### Uso

Para compilar el proyecto, ejecutar:

* Desde consola:

```bash
gprbuild -p -gnatef main.gpr
```

* Desde Visual Studio Code:

```bash
Ctrl + Shift + B
```

Para ejecutar el proyecto, ejecutar:

```bash
./objs/main
```
